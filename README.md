# AI-Work-Productivity-Assistant
A modern AI-powered productivity platform that helps workers to automate workplace tasks, improving efficiency and daily workflows.

## Project Overview

AI Workplace Productivity Assistant is a modern, AI-powered web application designed to help professionals automate common workplace tasks and improve productivity. The platform combines multiple AI-driven tools into a single dashboard, allowing users to generate professional emails, summarize meeting notes, plan tasks, conduct research, and interact with an intelligent workplace assistant.

The goal of the project is to demonstrate practical AI implementation, prompt engineering, real-world problem solving, responsible AI usage, and modern UI/UX design.

## Features

### Smart Email Generator

* Generate professional emails instantly
* Support for different tones (Formal, Friendly, Persuasive, Professional)
* Editable output
* Copy-to-clipboard functionality

### Meeting Notes Summarizer

* Summarize lengthy meeting notes and transcripts
* Extract key decisions
* Identify action items
* Highlight deadlines and important points

### AI Task Planner

* Create daily and weekly schedules
* Prioritize tasks based on urgency and importance
* Generate productivity recommendations

### AI Research Assistant

* Summarize topics, reports, and articles
* Provide key insights and recommendations
* Save research summaries for future reference

### AI Chatbot Interface

* Interactive workplace assistant
* Answer user questions
* Provide productivity support and guidance
* Maintain conversational interactions

### Additional Features

* Responsive dashboard design
* Sidebar navigation
* Light and dark mode
* Modern SaaS-inspired user interface
* Responsible AI disclaimer

## Setup Instructions

### Prerequisites

Ensure you have the following installed:

* Node.js
* npm or yarn
* Git

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Parencia54/AI-Work-Productivity-Assistant.git
```

2. Navigate to the project folder:

```bash
cd AI-Work-Productivity-Assistant
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser and visit:

```text
http://localhost:5173
```
